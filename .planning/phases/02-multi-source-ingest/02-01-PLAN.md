# Plan 02-01: Reddit JSON Source Adapters (Wave 1)

**Phase**: 2 — Multi-Source Ingest
**Mode**: mvp (vertical slice — one new source family end-to-end)
**Depends on**: Phase 1 complete

## Goal
Add a `RedditJsonSource` adapter that satisfies `SourcePort`, parameterized by subreddit. Register four instances in `build_sources()`: r/all, r/business, r/retail, r/BuyItForLife. After this plan, `crawler run-once` fetches HN + 4 Reddit feeds in one run.

## Why one adapter, four instances
All four subs use the same endpoint shape (`/r/{sub}/hot.json`) and the same response schema. Writing one parameterized class avoids 4× copy-paste. Each instance gets its own `name` (`reddit_all`, `reddit_business`, etc.) so `topic_sources.source_name` correctly attributes per-sub.

## Tasks

### T01 — Reddit adapter (TDD: tests first)
- **Step 1 — write the failing test first**: file `services/crawler/tests/test_reddit_adapter.py`
  - Mock `httpx.AsyncClient` with `httpx.MockTransport` returning a 4-item Reddit JSON fixture (3 valid + 1 with empty title).
  - Test `test_fetch_returns_three_items_with_correct_ranks`: expects ranks 1/2/4 (gap preserved when item 3 is dropped for empty title).
  - Test `test_self_post_uses_permalink`: an item where `data.url` is a Reddit-internal `/r/.../comments/...` link must produce a `RawItem.url` that starts with `https://www.reddit.com`.
  - Test `test_link_post_uses_external_url`: an item with a normal `data.url` must keep that URL.
  - Test `test_source_name_matches_constructor`: `RedditJsonSource(name="reddit_bifl", subreddit="BuyItForLife")` produces items with `source_name == "reddit_bifl"`.
  - Test `test_user_agent_header_set`: assert the outgoing request used the `Trend-Researcher/0.1 ...` UA (capture via MockTransport handler).
  - Run `uv run --package crawler pytest tests/test_reddit_adapter.py -v` → all FAIL (file under test does not exist yet). Confirm RED.
- **Step 2 — implement to green**: file `services/crawler/src/crawler/adapters/sources/reddit.py`
  - Class `RedditJsonSource(name: str, subreddit: str, http_client=None)`:
    - `name` is the public attribute satisfying `SourcePort` (e.g. `"reddit_bifl"`).
    - `subreddit` is the URL slug (e.g. `"BuyItForLife"`).
    - `fetch(top_n)` GETs `https://www.reddit.com/r/{subreddit}/hot.json?limit={top_n}` with `User-Agent: Trend-Researcher/0.1 (internal PoC; +https://github.com/thbe/trend-researcher)`.
    - Parse `data.children[*].data` into `RawItem`:
      - `title` = `data.title`
      - `url` = `data.url` if present and not a Reddit-internal `/r/.../comments/...` permalink, else `https://www.reddit.com` + `data.permalink`
      - `native_rank` = enumerate from 1
      - `observed_at` = `datetime.now(timezone.utc)` at fetch time
      - `raw_payload` = the per-child `data` dict
    - Skip items where `title` is empty or missing (log warning, don't renumber — match HN behavior).
    - 10s timeout. Owns its `httpx.AsyncClient` if none injected (mirror HN pattern).
- **Step 3 — verify GREEN**: `uv run --package crawler pytest tests/test_reddit_adapter.py -v` → all 5 tests pass.
- Commit: `feat(02-01-T01): RedditJsonSource adapter (TDD: 5 tests written first, then impl)`
